# Tobi's Workshop


I'm learning Kubernetes and DevOps by building a homelab that reflects how things are done in the real world.
The focus is on best practices, standards, and patterns used at enterprise and production level;
not just making things work, but making them work correctly.

## What I'm learning
- Container orchestration with **Kubernetes**
- Infrastructure provisioning with **Terraform**
- Configuration management with **Ansible**
- General **DevOps** practices and tooling

## What you'll find here
- **[kubernetes-homelab](../../../kubernetes-homelab/)** — My Kubernetes cluster setup: GitOps, secret encryption, automated updates
- **[homelab-infra](../../../homelab-infra/)** — Infrastructure as Code for provisioning and configuring VMs and LXCs using Terraform and Ansible
- **[lab](../../../lab/)** — Smaller experiments, tests, and learning projects
